gentlr

A RAM-positive macOS process optimizer with real PCA, clustering, SVM, RandomForest, GradientBoosting, XGBoost, and a tiny floating organism UI.

Download macOS GitHub Releases

Real ML

No mock scorers. The shipped pipeline uses sklearn/XGBoost models and stores a trained bundle locally.

One Supervisor

Watch up to 30 app policies from one shared process instead of launching 30 RAM-hungry agents.

Gentle Apply

Dry-run by default. Closing requires explicit apply mode and high-confidence safety thresholds.

Install

curl -L -o gentlr.zip https://github.com/overandor/gentlr/releases/latest/download/gentlr-v0.1.0-macos-arm64.zip unzip gentlr.zip cd gentlr ./install.sh ./gentlr-ui
