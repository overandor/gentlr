# gentlr v0.1.0

Production-ready first release for macOS Apple Silicon.

## What Ships

- Real ML process optimizer using sklearn and XGBoost.
- Real PCA and KMeans clustering.
- Shared supervisor for up to 30 app policies.
- Native Swift/AppKit floating microorganism widget.
- Menu bar controls for train, refresh, and safe one-at-a-time apply.
- Dry-run default with explicit `--apply` requirement.
- Strong safety rules for main apps, flow tools, and system daemons.

## Install

```sh
./install.sh
```

## Run

```sh
./gentlr-ui
```

## Notes

`gentlr` is RAM-positive by design: one shared supervisor watches all app policies instead of launching one ML runtime per app.
